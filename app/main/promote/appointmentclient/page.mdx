## Steps:

<h2 className="text-2xl font-semibold text-gray-800 mt-8 mb-4">Step-by-step Instructions</h2>

❶ **Go to the “Promote” section from the main page.**

❷ **Open the “Message Center” tab.**

❸ **Select the option titled “Appointment Updates – Client.”**

❹ **Here, messages will be configured for events such as Cancel, Reschedule, Confirmations, Reminders, and Review Requests.**

❺ **Use toggles to enable or disable these messages as needed.**

❻ **To view or customize the message content, use the preview/template button.**

---

**Tip for First-Time Users**:  
Keeping these automated updates on helps reduce no-shows and keeps clients informed in real-time.
