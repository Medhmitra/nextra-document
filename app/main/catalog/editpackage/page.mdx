## Steps:

<h2 className="text-2xl font-semibold text-gray-800 mt-8 mb-4">Step-by-step Instructions</h2>

‚ù∂ **Navigate to the Packages tab in the Catalog**  

‚ù∑ **Click on the three-dot menu of the package you want to update or open the package detail view and use the top menu**  

‚ù∏ **Make necessary edits in Basic Info, Service Selection, Pricing, Venues, or Staff Assignments**  

‚ùπ **Click Save to apply changes**  

‚ù∫ **A success message confirms the update**  

---

**Tip for First-Time Users**:  
Make sure to double-check all changes before saving, especially when updating pricing or staff.
