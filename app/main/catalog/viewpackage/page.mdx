## Steps:

<h2 className="text-2xl font-semibold text-gray-800 mt-8 mb-4">Step-by-step Instructions</h2>

‚ù∂ **Click on any package name from the Package List**  
To open the detailed view.

‚ù∑ **Review package information**  
The panel shows the **Service Inclusions**, **Pricing Breakdown**, **Venue**, and **Staff** details.

‚ù∏ **Return to list**  
Use the **Close/Cancel** icon to return.

---

**Tip for First-Time Users**:  
Reviewing package details is useful before making edits or confirming bookings.
