*Purpose*: Learn how to configure and publish a discount code through the Offers section.

## Steps:



‚ù∂ **Navigate to the Offers section**  
Click on **Offers** in the sidebar.

‚ù∑ **Click the Promo Code tab**  
Inside Offers, click the **Promo Code** tab.

‚ù∏ **Click Add Promo Code**  
Hit the **Add Promo Code** button at the top-right of the screen.

‚ùπ **Fill in the Promo Code details**  
A panel form will open. Complete the fields:
- **Name**, **Description**, and **Discount Code**
- **Start Date**, **End Date**, and **Active Days**

‚ù∫ **Select Services**  
Click **Next** to choose the **Services** the promo code applies to.

‚ùª **Define Value and Limits**  
Click **Next** again to configure:
- **Value Type** (Percentage or Amount)
- **Value**
- **Limitations** like:
  - One use per client
  - Total number of uses
  - Minimum purchase amount
  - Off-peak availability

‚ùº **Configure Booking Options**  
Click **Next** to choose where the code applies: **Online**, **Walk-in**, or both.

‚ùΩ **Assign Venues**  
Click **Next** and select the **Venue(s)** the code applies to.

‚ùæ **Click Save**  
Finish setup by clicking the **Save** button.

---

**Tip for First-Time Users**:  
Using promo codes strategically can boost engagement and sales, especially during campaigns or slow periods.
