*Purpose*: Remove a promotional code that is no longer needed from the Offers section.

## Steps:

<h2 className="text-2xl font-semibold text-gray-800 mt-8 mb-4">Step-by-step Instructions</h2>

‚ù∂ **Navigate to Offers > Promo Code**  
Go to the **Offers** section in the left sidebar and select **Promo Code**.

‚ù∑ **Click the 3-dot Menu**  
For the promo code you wish to delete, click on the **3-dot menu** beside it.

‚ù∏ **Confirm Deletion**  
Select **Delete**, and confirm the action in the **pop-up prompt** to permanently remove the promo code.

---

**Tip for First-Time Users**:  
Be cautious before deleting a promo code, as this action cannot be undone.
