## Steps:

<h2 className="text-2xl font-semibold text-gray-800 mt-8 mb-4">Step-by-step Instructions</h2>

❶ **Go to Offers > Promo Code**  
Navigate to the **Offers** section from the sidebar and click on **Promo Code**.

❷ **Click on the Promo Code Name**  
Select the specific promo code whose details you want to view.

❸ **Review All Details**  
Check the following sections:
- **Basic Info**
- **Services** the promo applies to
- **Venues** where it’s valid
- **Client Usage** stats and history

❹ **Click the Cancel Icon to Return**  
Use the **Cancel** icon or button to go back to the Promo Code listing page.

---

**Tip for First-Time Users**:  
Reviewing promo code details helps track campaign performance and usage across services and clients.
